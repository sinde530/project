# CoodSoom 복습하기

- npx serve 를 이용하여 index.html 로컬에 보이게 하기.
- webpack 설치하기.

```typescript
"dependencies": {
    "webpack": "^5.73.0",
    "webpack-cli": "^4.9.2",
    "webpack-dev-server": "^4.9.2"
  }
```

- npx webpack-dev-server 사용해보기
- eslint install 후 초기화 시키기
   - `yarn eslint --init`
- eslint 커맨드 사용해서 틀린부분 맞쳐주기.
   - `yarn eslint --fix .`

- public => index.html 넣기
- index.js innerHTML사용하여 글자 출력하기
